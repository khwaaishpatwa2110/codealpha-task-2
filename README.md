# 🎵 ARIRANG – Music Player

A clean, responsive music player web app built as part of the **CodeAlpha Internship – Task 2**.

🔗 **Live Demo:** [khwaaishpatwa2110.github.io/codealpha-task-2](https://khwaaishpatwa2110.github.io/codealpha-task-2/)

---

## 📸 Preview

![ARIRANG Music Player](arirang_album_cover.jpg)

---

## ✨ Features

- 🎶 Tracklist display with song and artist info
- ▶️ Play, pause, and track navigation controls
- ⏱️ Real-time progress bar with current time display
- 🖼️ Album cover art display
- 📱 Responsive design — works on mobile and desktop

---

## 🛠️ Built With

- **HTML5** – Structure, styling, and player logic all in a single file
- **CSS3** – Embedded styles for layout and responsiveness
- **JavaScript** – Embedded script for player controls and audio handling
- **Web Audio API** – Native `<audio>` element for playback

---

## 🚀 Getting Started

To run this project locally:

```bash
git clone https://github.com/khwaaishpatwa2110/codealpha-task-2.git
cd codealpha-task-2
```

Then open `index.html` in your browser — no build step or dependencies required.
